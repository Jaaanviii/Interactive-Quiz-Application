As part of my internship, Task 1 involved designing and developing a Quiz Web Application using JavaScript. The goal was to create a polished, interactive quiz interface where users could answer multiple-choice questions and receive instant feedback and scoring upon completion. This task helped me practice core frontend development skills such as DOM manipulation, event handling, array management, and dynamic content rendering, all in a practical, engaging context.

The deliverable was a fully functional and styled web-based quiz that supports dynamically loaded questions, user interaction, score tracking, and immediate result display after the user finishes the quiz.

The objective of this task was to simulate a real-world online quiz system, like those used in e-learning platforms, mock tests, or training modules. It had to be built using vanilla HTML, CSS, and JavaScript, without relying on external frameworks or libraries. The final product aimed to offer a simple yet powerful learning experience with responsive design and clear user feedback.

This project was also meant to reinforce important programming concepts like functions, loops, conditional logic, and event listeners, while ensuring that the interface remained user-friendly and visually appealing.

üìê Application Structure

The quiz application consists of the following key components:

1. Start Screen
   
Displays the quiz title and a ‚ÄúStart Quiz‚Äù button

On click, transitions to the first question

2. Question Display SectioN
   
Dynamically displays one question at a time

Each question includes:

A question title

Multiple-choice options (radio buttons or clickable buttons)

Clicking on an answer triggers validation and immediate visual feedback (e.g., green for correct, red for incorrect)

3. Next Button
   
After selecting an answer, a "Next" button appears to proceed to the next question

Prevents moving forward unless an answer is selected

4. Score Display
   
Once all questions are answered, the final score is displayed

Shows how many questions were answered correctly

Option to retake the quiz or return to home screen

5. Dynamic Question Loading
   
Questions are loaded from an array of objects
